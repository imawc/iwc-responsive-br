IWC RESPONSIVE BR uses the Format API to add a responsive BR.

== Description ==

IWC RESPONSIVE BR adds a BR tag to the gutenberg toolbar.

= GitHub =

https://github.com/imawc/iwc-responsive-br

== Installation ==

1. Upload `iwc-responsive-br` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
